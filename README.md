# Мод "Блокировка управления мышью"

| [**Download / Скачать**](https://github.com/litolax/supercow-block-mouse-control/releases/latest/download/block-mouse-control.zip) |
|-------------------------------------------------------------------------------------------------------|

Данный мод для игры "Суперкорова" позволяет заблокировать управление с помощью мыши и переключиться полностью на клавиатурное управление.

Сделан на базе [SuperMod](https://github.com/zziger/supercow-mod)

## Инструкции по установке

1. Скачайте и установите [SuperMod](https://github.com/zziger/supercow-mod)
2. Скачайте последнию версию [мода](https://github.com/litolax/supercow-block-mouse-control/releases/latest/download/block-mouse-control.zip)
3. Перетяните архив с модом на окно игры
